# AI Resume Keyword Matcher

A simple AI-inspired Resume Analyzer web app built using Python and Streamlit.

## Features

- Upload Resume PDF
- Paste Job Description
- Match Percentage
- Missing Keywords
- Simple UI

## Technologies Used

- Python
- Streamlit
- PyPDF2
- Scikit-learn

## Installation

```bash
pip install streamlit PyPDF2 scikit-learn
```

## Run Project

```bash
streamlit run app.py
```
